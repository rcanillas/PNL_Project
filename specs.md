Les plus importants: 
- Tri sur soi / tri sur l'autre: tri sur soi = penser à ses propres besoins en premier ("j'ai pensé que ça serait mieux qu'on parte ce weekend"/"je préfère ..."). Tri sur l'autre = faire en sorte que l'autre soit satisfait.
- Aller vers / éviter de: Aller vers = avoir un objectif et aller vers lui, éviter de = éviter les choses négatives. 
- Actif / passif: passif = les choses m'arrivent / actif = je fais bouger les choses.
- Référence interne / référence externe: référence interne = se référer à sa propre vision du monde / ref externe = se placer dans la vision de quelqu'un d'autre.
